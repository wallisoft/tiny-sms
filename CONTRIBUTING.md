# Contributing to Tiny-SMS

PRs welcome! Keep it simple — this app is intentionally minimal.

## Ideas for contributions
- Settings screen for configurable poll interval
- Multiple Gmail account support
- Notification when SMS is sent/received
- Contact name lookup
- Dark/light theme toggle
- Translations

## Code style
- Standard Android Java conventions
- 200 lines max per file (seriously, split it if it grows)
- Comment anything non-obvious

## Submitting
1. Fork the repo
2. Create a feature branch
3. PR against `main` with a clear description
